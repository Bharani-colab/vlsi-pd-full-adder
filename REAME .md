# Full Adder Physical Design Project

## About Project
This project shows basic VLSI Physical Design flow using Full Adder circuit.

## Inputs
A, B, Cin

## Outputs
Sum, Cout

## PD Flow Steps
1. RTL Design (Verilog coding)
2. Logic Synthesis
3. Floorplanning
4. Placement
5. Routing
6. Layout generation

## Tools
OpenROAD (Physical Design Tool)
